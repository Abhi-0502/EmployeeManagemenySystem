#Employee Management System – Complete Project Overview
Project Title
Employee Management System using Spring Boot, JWT Authentication, Swagger, MySQL, Docker, and REST APIs

1. Project Overview

The Employee Management System is a secure full-stack web application developed using Spring Boot and Spring Security. The application allows authenticated users to manage employee records with complete CRUD operations such as adding, updating, viewing, and deleting employees.

JWT Authentication is implemented to secure REST APIs and provide authorized access to users. Swagger is integrated for API documentation and testing. The project follows layered architecture using Controller, Service, Repository, Security, and Entity layers for better maintainability and scalability.

Docker support is added for containerized deployment and easy project setup across environments.
